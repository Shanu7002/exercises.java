
Exercícios de Java - Variáveis, Tipos de Dados, Operadores, Estruturas de Controle, Funções, Estruturas de Dados, I/O, Strings, OOP

=====================
1) Variáveis, Tipos de Dados e Operadores
=====================
1. Crie um programa que declare uma variável inteira, atribua um valor e exiba na tela.
2. Declare variáveis para armazenar seu nome, idade e altura, e exiba-os.
3. Converta um valor em metros para centímetros.
4. Calcule a área de um retângulo usando variáveis base e altura.
5. Faça um programa que receba dois números e exiba a soma, subtração, multiplicação e divisão.
6. Calcule o resto da divisão de dois números.
7. Use operadores de incremento e decremento e observe o resultado.
8. Verifique se um número é par ou ímpar.
9. Converta um número inteiro em double e vice-versa.
10. Calcule o IMC de uma pessoa.

2) Estruturas de Controle: if, for, while

15. Faça uma tabuada de um número informado pelo usuário.
16. Verifique se uma pessoa pode votar com base na idade.
17. Calcule a soma dos números de 1 a N usando for.
18. Conte quantos números entre 1 e 100 são divisíveis por 3.
19. Imprima a sequência Fibonacci até o 10º termo.
20. Crie um menu interativo usando switch-case.
